# House Price Prediction

## Overview

This project predicts residential house prices using Machine Learning techniques. The dataset was cleaned, analyzed, and transformed before training predictive models.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Models

* Linear Regression
* Random Forest Regressor

## Results

| Model                   | R² Score |
| ----------------------- | -------- |
| Linear Regression       | 0.64     |
| Random Forest Regressor | 0.887    |

## Key Findings

* OverallQual was the most important feature.
* Living Area and Basement Size strongly influenced house prices.
* Random Forest significantly outperformed Linear Regression.
